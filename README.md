# yoyo-solveAI

An AI-blog generator, utilize ChatGPT Create Subscription-based SAAS with MERN and Stripe

Backend: Node.js

OpenAI: chatGPT

Frontend: React

Subscription base: stripe

Database: MongoDB

Using OpenAI text generation models
